ℹ️ Gmail API Integration Setup

  1. Enable the Gmail API in your Google Cloud Console.
  2. Generate OAuth 2.0 Web Application credentials. Add redirect URI: https://developers.google.com/oauthplayground.
  3. Go to the Google OAuth Playground, enter the Client ID and Secret in settings (gear icon), select the https://www.googleapis.com/auth/gmail.send scope, and authorize.
  4. Exchange the authorization code for tokens, copy the Refresh Token, and paste it here.
  5. The system will encrypt secrets using AES-GCM and store them in the database vault.
